# IND-005: Calculator received a DataFrame without a required column

## Why this error fires

`_require_columns` (in `echolon/indicators/calculators/_utils.py`) is
called by intraday calculators to enforce their input contract at entry.
Each calculator declares the columns it *must* see (e.g. `trading_date`,
`datetime`, `volume`, `session_phase`). If any are absent, IND-005 is
raised before the calculator does any work — silent all-NaN or junk output
would otherwise propagate downstream without a trace.

Context dict:

- `calculator` — the `__name__` of the calculator module that raised
- `missing_column` — the first missing column detected
- `required_columns` — the full list the calculator declared
- `present_columns` — what the input DataFrame actually had

## Fix

Run the input DataFrame through the upstream transformer(s) that produce
the missing column before passing it to the calculator. The two most
common missing inputs:

- `trading_date` — produced by `calendar_generator` and the session-phase
  tagging step.
- `session_phase` / `session_phase_agg` — produced by
  `calculators.intraday.market_context.attach_session_phase`.

## Example

Triggering code (calling a VWAP calculator on raw OHLCV without running
the session-tagging step first — raw OHLCV has `date`/`datetime` but not
`trading_date`):

```python
from echolon.indicators.calculators.intraday.indicators import calculate_vwap
import pandas as pd

raw_ohlcv = pd.read_csv("workspace/data/market_data/SHFE/aluminum/1m.csv")
# raw_ohlcv columns: ['datetime', 'open', 'high', 'low', 'close', 'volume']
# missing 'trading_date' which is needed to group VWAP by session

calculate_vwap(raw_ohlcv)  # raises IND-005
# context: calculator='echolon.indicators.calculators.intraday.indicators',
#          missing_column='trading_date',
#          required_columns='trading_date',
#          present_columns='datetime, open, high, low, close, volume'
```

Corrected version (attach session context first):

```python
from echolon.indicators.calculators.intraday.market_context import (
    attach_session_phase,
)
from echolon.indicators.calculators.intraday.indicators import calculate_vwap

df = attach_session_phase(raw_ohlcv)     # adds datetime, trading_date, session_phase
df = calculate_vwap(df)                  # now succeeds
```
