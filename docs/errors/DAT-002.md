# DAT-002: State file is corrupt or unreadable JSON

## Why this error fires

A live deploy reads strategy_state.json to resume position and cycle counters. A truncated or malformed file silently defaults to an empty state, losing position information mid-session.

## Fix

Inspect the state file and either repair it or delete it to cold-start:
