# LIV-002: Order rejected by broker

## Why this error fires

The broker rejected an order. Common causes: price outside the day's range, insufficient margin, invalid contract code, direction/size mismatch.

## Fix

Inspect the rejected order and broker response:
  contract:       {contract}
  direction:      {direction}
  price:          {price}
  size:           {size}
  broker_status:  {broker_status}
  broker_message: {broker_message}
