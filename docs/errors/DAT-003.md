# DAT-003: Main contract data file not found for instrument

## Why this error fires

Echolon resolves the main contract per trading date from raw_data_dir/{exchange}/{symbol}/main_contract.csv. Without this file, contract rollover and live trading cannot proceed.

## Fix

Run the data pipeline once to populate main_contract.csv, or pass an explicit raw_data_dir pointing at a populated tree.
  expected:  {path}
  symbol:    {symbol}
