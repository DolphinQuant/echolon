# BT-003: Optuna trial violated a hard constraint

## Why this error fires

The trial's param set produced regime metrics outside the viability bounds configured in RegimeOptimizerConfig. The trial's score is clamped to 0.0 so it will not be selected.

## Fix

Widen the constraint or the param range that triggered this:
  trial_number:   {trial_number}
  constraint:     {constraint}
  required:       {required}
  actual:         {actual}
  params:         {params}
