# DAT-004: Trading calendar is empty after generation

## Why this error fires

Calendar generation received zero valid rows. Either the input data has no date column, all dates are outside the requested range, or the source file is empty.

## Fix

Verify the upstream source has dated rows in the requested range:
  market:      {market}
  instrument:  {instrument}
  start_date:  {start_date}
  end_date:    {end_date}
  rows_seen:   {rows_seen}
